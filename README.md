# Lending Club Loan Data (2007-2020 Q3)

## Dataset Source

This project uses the Lending Club loan data from Kaggle. The dataset can be downloaded using the following code:

```python
import kagglehub

# Download latest version
path = kagglehub.dataset_download("ethon0426/lending-club-20072020q1")
```

## About the Dataset

The dataset contains Lending Club loan data spanning from 2007 to Q3 2020, sourced from [Kaggle](https://www.kaggle.com/datasets/ethon0426/lending-club-20072020q1).

## Setup

To download the dataset, you'll need to:

1. Install the kagglehub package:
```bash
pip install kagglehub
```

2. Authenticate with your Kaggle credentials
